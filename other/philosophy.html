What is up with this site?

If you are wondering why this site looks like this then you have come to the right page!

This site follows a minimalist philosophy. Meaning this site doesn't use any bloated frontend languages and just sticks to mainly html & css only. I also stay away from javascript (using no javascript at all on this site) as I'd like to keep my website functional by default for people that use javascript blockers for privacy reasons or people that are using tor. I like to keep it simple (it saves a lot of headache and pages load fast!).

Anyway... That is why my page looks like an old-school shit. Cool?
